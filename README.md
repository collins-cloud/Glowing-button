# Glowing-button
A simple glowing button made with HTML and css. This project is great for beginners learning animations, transition, and UI design.

# Features

Smooth glow animation

Pure HTML & CSS (no JavaScript)

Customizable colors and glow intensity

Beginner-friendly code structure

# Technologies Used

HTML5

CSS3 (animations, box-shadow, transitions)
